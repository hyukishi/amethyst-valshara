Why this layout

            
This client keeps the terminal central, gives the map its own readable space, and surfaces clickable movement so the browser experience still feels like a MUD client instead of a plain telnet box.

            
It also keeps account and character creation in the browser through the game’s live login flow, so the UI stays aligned with your real server logic.
